Escrow Release Tool provided by SBC

This tool allows you to release any time-based escrow created by Secure Block Chains

To use this tool you need 4 simple pieces of information. The Secret Key is required to pay the Network Fee.

  • The Secret Key to any XRP wallet (Master or Regular).
  • The Public Address that Corresponds to that Secret Key
  • The Transaction Hash of the Escrow you wish to unlock.
  • The Fulfillment ID for the Escrow you wish to unlock. (Or Escrow Password If You Used Harbor + Escrow Password)
  • (Optional)The default fee the app uses is 5000 drops or 0.005 XRP if your transaction is rejected you can increase this fee. You must enter the fee amount in DROPS. There are 1 Million drops in 1XRP.
